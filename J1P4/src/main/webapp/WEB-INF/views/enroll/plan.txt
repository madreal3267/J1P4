1. 임시저장 기능 구현
-> ajax 활용 (참고 링크 : https://cbn1218.tistory.com/15)

2. 임시저장 구현 상태
-> 데이터베이스에 연결되어 값이 저장은 되나 null로 저장되고 있음
   form 데이터 전달이 안 되고 있는 상황
   
3. 임시저장 했을 때 단순히 insert만 되면 안되고 temporary 컬럼에
   임시저장여부를 update 해줘야함 -> 즉 Controller에서 매핑을 다르게 해줘서
   update 쿼리도 실행되게 하거나 insert 쿼리에 temporary 컬럼도 넣어야함

4. 정리 : ajax를 이용한 DB로 데이터 전달이 성공적으로 되면
		  매핑을 다르게 해줘서 데이터 전달이 되나 확인할 것
